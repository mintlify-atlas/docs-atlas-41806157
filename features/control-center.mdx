The Control Center is a unified popout panel that surfaces the most common system controls in one place: network connections, Bluetooth devices, audio inputs and outputs, display brightness, and night mode. It opens from the DankBar or from any IPC call, and closes when you click outside or press Escape.

## Opening the control center

```bash
# Toggle the control center open/closed
dms ipc call control-center toggle

# Open or close explicitly
dms ipc call control-center open
dms ipc call control-center close
```

The control center button in DankBar's right section opens it by default.

## Network

The network panel handles both Wi-Fi and wired connections:

- **Wi-Fi scanning** — triggers an active scan for available access points
- **Connecting** — select a network and enter a password to connect
- **VPN** — shows active VPN connections; the VPN widget is a built-in control center plugin

NetworkManager is used as the primary backend. A fallback legacy backend is available for environments without NetworkManager.

## Bluetooth

The Bluetooth panel handles device discovery and pairing:

- **Discovery** — toggles the adapter scan mode to find nearby devices
- **Pairing** — initiates pairing with a discovered device
- **Connection** — connects and disconnects paired devices
- **Device types** — icons indicate the device category (headphones, keyboard, phone, etc.)

D-Bus integration is handled via `org.bluez` with a built-in pairing agent.

## Audio

The audio panel controls:

- **Output volume** — a slider for the current output device's volume
- **Input volume** — a separate slider for microphone level
- **Device selection** — choose between available output and input devices
- **Mute** — toggle mute on output or input independently

## Display brightness

A brightness slider is available for displays that expose a backlight or support DDC/CI. Multiple monitors are handled separately.

```bash
# Adjust brightness via IPC
dms ipc call brightness set 80
dms ipc call brightness increment 10
dms ipc call brightness decrement 5 "intel_backlight"

# List available brightness devices
dms brightness list
```

## Night mode

Night mode reduces the color temperature of your display to warmer tones, easing eye strain in low-light conditions. It is controlled via the `night` IPC target and configured in the control center panel.

### Basic controls

```bash
# Toggle night mode on or off
dms ipc call night toggle

# Enable or disable explicitly
dms ipc call night enable
dms ipc call night disable

# Check current state
dms ipc call night status
```

### Color temperature

```bash
# Get the current temperature
dms ipc call night temperature

# Set temperature in Kelvin (2500–6000, steps of 500)
dms ipc call night temperature 4000
```

### Automation

Night mode can activate automatically based on a schedule or your geographic location.

```bash
# Set automation mode: manual, time, or location
dms ipc call night automation manual
dms ipc call night automation time
dms ipc call night automation location

# Configure a time-based schedule (HH:MM format)
dms ipc call night schedule 20:00 06:00

# Set coordinates for location-based automation
dms ipc call night location 40.7128 -74.0060
```

<AccordionGroup>
  <Accordion title="Manual mode">
    Night mode only activates or deactivates when you explicitly call `enable`, `disable`, or `toggle`. No automatic scheduling.
  </Accordion>
  <Accordion title="Time-based mode">
    Night mode activates at the `start` time and deactivates at the `end` time set by `schedule`. Uses 24-hour format.
  </Accordion>
  <Accordion title="Location-based mode">
    Night mode follows local sunrise and sunset times calculated from the coordinates set with `location`. The shell resolves your location automatically if a location service is available, or you can set coordinates manually.
  </Accordion>
</AccordionGroup>

## Do Not Disturb

The control center includes a Do Not Disturb toggle that suppresses notification popups while active. Notifications are still recorded in the notification center and become visible when DND is turned off.
