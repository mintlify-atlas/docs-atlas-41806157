DankMaterialShell (DMS) is a complete desktop environment for Wayland compositors. It replaces the patchwork of separate tools — waybar, swaylock, swayidle, mako, fuzzel, and polkit — with a single, cohesive shell built on [Quickshell](https://quickshell.org/) and Go. Dynamic wallpaper-based theming, a powerful application launcher, a unified control center, and an extensible plugin system are all included out of the box.

## Get up and running

<Steps>
  <Step title="Install DankMaterialShell">
    Run the one-line installer — it handles dependencies, compositor config, and first launch:
    ```bash
    curl -fsSL https://install.danklinux.com | sh
    ```
  </Step>
  <Step title="Choose your compositor">
    DMS works with Niri, Hyprland, Sway, MangoWC, labwc, Scroll, and Miracle WM. Follow the [compositor guide](/compositors) to set up keybinds and workspace integration.
  </Step>
  <Step title="Set your wallpaper and theme">
    Set a wallpaper and DMS automatically generates a matching color scheme for your entire desktop:
    ```bash
    dms ipc call wallpaper set ~/Pictures/my-wallpaper.jpg
    ```
  </Step>
  <Step title="Explore features and plugins">
    Browse the [plugin registry](https://plugins.danklinux.com) to add bar widgets, launcher extensions, and desktop widgets. Read the [features overview](/features/theming) for everything included by default.
  </Step>
</Steps>

## Supported compositors

DMS supports the following Wayland compositors with full workspace switching, overview integration, and monitor management:

| Compositor | Workspaces | Overview | IPC |
|---|---|---|---|
| [Niri](https://github.com/YaLTeR/niri) | Full | Yes | Yes |
| [Hyprland](https://hyprland.org/) | Full | Yes | Yes |
| [Sway](https://swaywm.org/) | Full | — | Yes |
| [MangoWC](https://github.com/DreamMaoMao/mangowc) | Tags | — | dwl-ipc |
| [labwc](https://labwc.github.io/) | Full | — | Yes |
| [Scroll](https://github.com/dawsers/scroll) | Full | — | Yes |
| [Miracle WM](https://github.com/miracle-wm-org/miracle-wm) | Full | — | Yes |

Other Wayland compositors work with reduced feature support.
